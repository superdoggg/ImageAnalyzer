# ImageAnalyzer

MFC dialog-based test program for loading a sample image and a target image.

## Current Scope

- Load `sample.png` and `target.png` from the workspace when available.
- Let the user choose sample/target image files manually.
- Display both images in the dialog.
- OpenCV and image matching are intentionally not included yet.

## Environment

- Visual Studio Community 2026
- MFC/ATL workload installed
- Windows 11

## Build

Open `ImageAnalyzer.sln` in Visual Studio and build the `x64` configuration.
